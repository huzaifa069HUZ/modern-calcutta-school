# Modern Calcutta School — Homepage

A premium, high-end homepage for an Islamic school blending modern international education with refined cultural identity.

## Project Structure

```
modern calcutta/
├── index.html          # Complete homepage with 7 sections
├── style.css           # Full design system & responsive styles
├── main.js             # Scroll animations & interactions
└── assets/             # Images directory (placeholders active)
```

## Design Features

- **Hero Section:** Full-width background with gradient overlay
- **About Preview:** Split layout with elegant typography
- **Why Choose Us:** 4-card grid with custom SVG icons
- **Faith & Future:** Split section with vertical divider
- **Gallery Preview:** 3×2 grid with hover scale effects
- **Admissions CTA:** Centered call-to-action with geometric watermark
- **Footer:** Dark warm theme with crescent watermark

## Color Palette

- Primary BG: `#F5EDE3` (soft peach-beige)
- Structure Blue: `#1B3A5C`
- Cream: `#FFF8F0`
- Muted Gold: `#C5A55A` (subtle accents)
- Emerald: `#2E7D5B` (micro-highlights)

## Typography

- **Headings:** Poppins (800-900 weight) — substitute for EPIC PRO
- **Body:** Inter
- **Urdu:** Noto Nastaliq Urdu

## Responsive Breakpoints

- Desktop: 1440px max-width
- Tablet: ≤1024px
- Mobile: ≤768px

## To Add Images

Place the following images in the `assets/` folder:
- `hero-bg.jpg` - Full-width hero background (1920×1080px)
- `about-school.jpg` - School building exterior
- `gallery-1.jpg` through `gallery-6.jpg` - Campus life photos

Then update the hero section CSS background-image property in `style.css`.

## Open in Browser

Simply open `index.html` in a modern browser to view the homepage.
