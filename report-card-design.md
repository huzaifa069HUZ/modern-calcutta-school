# Report Card Design Implementation

🤖 **Applying knowledge of `@[frontend-specialist]`...**

## 🎨 DESIGN COMMITMENT: MODERN APPLE-INSPIRED A4

- **Topological Choice:** An elegant digital layout optimized for beautiful physical printing. The design strictly uses the `a4-preview` standard dimensions (`210mm x 297mm`) forcing the desktop view to reflect exactly what the printer output will yield.
- **Risk Factor:** Balancing strict rigid table lines with pure Apple-inspired UI. We chose 1px `#E5E7EB` internal borders with alternating light rows (`#fcfcfc`) and a heavier border (`1.5px`) outside the table to frame it perfectly. 
- **Readability Conflict:** High-density data means limited space. We solved this using the `Inter` font for numbers/marks, `Montserrat` for strong geometric headers, and `Poppins` for body text. 
- **Cliché Liquidation:** The "Blue Trap" was managed by applying specifically `#1E3A8A` (Deep Royal Blue) offset by `#DBEAFE` instead of the typical Tailwind generic cyan or generic blue. Absolutely no purple or aurora gradients were used, matching the exact minimalist palette provided.

## Academic Structure Maintained
- Exact column names (1st Unit, Half Yearly, 2nd Unit, Final Term).
- Subject structures meticulously retained (English with Written, Oral, Rhymes, Hand Writing; Math with Written, Oral... etc).
- The DRAWING subject does not have sub-rows and allows entering marks on the same level.
- Totals (Marks Obtained, Grand Total...) are cleanly docked underneath identical to the paper report card but enhanced with visually striking external summary boxes.
